## 🛠️ Tech Stack & Architecture

The Curator is built purely as a high-performance, cross-platform mobile experience:

* **Frontend Framework:** Flutter & Dart — Leveraging a declarative UI engine to build responsive, pixel-perfect layouts for both iOS and Android.
* **State Management:** Provider — Used for clean, reactive state propagation across the authentication and user data flows.
* **Navigation Architecture:** GoRouter — Provides declarative, robust routing across screens (including deep-linking readiness).
* **Networking Engine:** Dio HTTP Client — Engineered with secure secure-storage token interceptors, automated Bearer token sanitization, and graceful status code validation guards.
